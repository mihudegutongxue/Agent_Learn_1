# 专注时钟 (Pomodoro Timer)

这是一个简洁、现代化的番茄钟应用，专为专注工作和学习设计。

## ✨ 特性

- **极简设计**: 采用玻璃拟态 (Glassmorphism) 风格，界面干净清爽。
- **三种模式**: 
  - 🔴 专注 (默认 25分钟)
  - 🟢 短休息 (默认 5分钟)
  - 🔵 长休息 (默认 15分钟)
- **可视化进度**: 优雅的环形进度条，直观感受时间流逝。
- **个性化设置**: 
  - 自定义每个模式的时长。
  - 开启/关闭桌面通知。
  - 配置自动保存 (LocalStorage)。
- **任务专注**: 支持输入当前专注的任务名称。
- **单文件**: 所有代码（HTML, CSS, JS, SVG）都在一个文件中，无需构建工具。
- **响应式**: 完美适配桌面和移动设备。

## 🚀 如何部署到 GitHub Pages

1. **创建仓库**: 在 GitHub 上创建一个新的仓库（例如 `pomodoro-timer`）。
2. **上传文件**: 将 `index.html` 文件上传到该仓库的根目录。
3. **开启 Pages**:
   - 进入仓库的 **Settings** (设置)。
   - 在左侧菜单找到 **Pages**。
   - 在 **Build and deployment** 下的 **Source** 选择 `Deploy from a branch`。
   - 在 **Branch** 下选择 `main` (或 `master`) 分支，文件夹选择 `/ (root)`。
   - 点击 **Save**。
4. **访问**: 等待几分钟后，刷新页面，你会看到顶部出现你的网站链接（例如 `https://yourusername.github.io/pomodoro-timer/`）。

## 💻 本地运行

直接双击打开 `index.html` 即可在浏览器中使用。

## ⌨️ 快捷键

- **空格 (Space)**: 开始/暂停计时
- **Esc**: 关闭设置弹窗
